
Location Detection with GPS

This is a PHP, HTML5 & JavaScript code for your website to be location informed!

This code use browser support for navigator.geolocation to get Latitude and Longitude values of your website visitors. Then by using these values get reverse geocode by querying Google Maps API with PHP.Then Parse the XML data from the Google Maps API. Update the page with the data.


You may use this code freely in any of your projects or websites.
